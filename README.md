# Tainan-trip

台南三天兩夜行程（11/08 - 11/10）的前端小專案，用來給同行朋友快速確認：
行程、地點、停車、注意事項跟大概費用，一打開就知道今天要去哪。

## 功能總覽

- 📅 **分頁式行程**
  - 依日期切換：`11/08（六）`、`11/09（日）`、`11/10（一）`、`總覽 & 備忘`
  - 單頁 SPA，適合手機＋GitHub Pages 瀏覽，不用捲一整長頁。

- 📍 **景點＋停車整合**
  - 每個行程卡片包含：時間、地點名稱、Google Maps 連結。
  - 提供建議停車場與簡單說明，方便自駕使用。

- 💰 **費用試算（固定版）**
  - 已包含：住宿、租車、桌遊、KTV、貓咖、Simple Plan 晚餐。
  - 依實際設定人數拆分：
    - 一般：5 人
    - 桌遊：6 人
    - KTV：7 人
    - 貓咖：6 人
    - 晚餐（Simple Plan）：6 人
  - 顯示每人區間金額（不含停車與其他餐飲），方便先抓預算。

- 🐱 **注意事項整理**
  - 呼嚕咖啡館 x 貓中途驛站（貓咖）入場規則。
  - 桌遊東安店使用規則（禁菸、禁外食、脫鞋、禁賭博相關遊戲）。
  - 適合直接截圖／轉傳給同行友人。

## 技術細節

- 純靜態頁面：`index.html`
- 使用 [Tailwind CSS CDN](https://tailwindcss.com/) 做版面與 RWD
- 原生 JavaScript 控制分頁切換與按鈕互動
- 無任何後端、無打 API，適合部署在：
  - GitHub Pages
  - Netlify / Vercel / 任意靜態空間
